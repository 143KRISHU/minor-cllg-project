Text Properties

  • text-align : left or start/right or end/center/justify.
  • font-weight : normal/bold/100/bolder/lighter
  • text-decoration : none/underline/overline/line-through
  • line-height : numeric value
  • letter-spacing : it controls the character space units are normal /10px
  • font-size :
    absolute units : px,pt,pc,cm,mm,in.
    Relative units : %,em,rem,ch,vh,vw + many more.
  • Font-family : specifies a priortized list of one or more font family names.
